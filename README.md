# Programa para resolver o exercício de vetores abaixo.

O exercício é:.
Uma escola possui N alunos, faça um algoritmo que armazene na memória todos os nomes destes alunos de tal
forma que nenhum nome seja perdido. A escola comporta no máximo 100 alunos. Exiba a lista de alunos depois da
leitura.
Desenvolvido utilizado JDK 1.8 e NetBeans 13.
